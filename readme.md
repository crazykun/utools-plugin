# utools-plugin

## utools插件

- encrpt: 加密解密插件

    `加密解密 是一款为utool开发的插件工具, 方便加密解密测试。`
- ip: ip归属地查询插件

    `utools ip归属地查询`
- var: 变量名生成插件

    `变量名生成插件`


### 使用方法

utools搜索
`加密`或`解密` 打开加密解密插件 | md5| sha1| sha256| RIPEMD160| AES| DES| RC4

`ip` 打开ip归属地查询插件
`var` 打开变量名生成插件
