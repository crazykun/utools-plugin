# utools插件

encrpt: 加密解密插件
ip: ip查询插件


## 加密解密
utools加密解密插件

### 工具介绍
加密解密是一款为utool开发的插件工具, 方便加密解密测试。


### 支持方法
- md5
- sha1
- sha256
- RIPEMD160
- AES
- DES
- RC4
